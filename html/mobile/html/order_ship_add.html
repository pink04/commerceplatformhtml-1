iOS Safari 브라우저 이용시 유의사항

  • iOS Safari 브라우저 보안이슈로 현재 국민/우리/전북/수협 카드사 결제가 불가능합니다.
  • iPhone 및 iPad에서 iOS 11.3 이상 업데이트 후 결제 오류 발생 시 아래와 같이 조치해주시기 바랍니다.

    설정 > Safari > 위조된 웹사이트 경고 > OFF 로 전환 위조된 웹사이트 경고가 OFF 인데도 결제가 되지 않는 경우, OFF → ON → OFF로 재설정

원더코인 사용안내

  • 원더코인은 1000 코인 이상부터 1코인 단위로 결제 시 사용 가능합니다.
  • 원더코인은 기본적으로 적립일로부터 1년 동안 사용 가능하며 이후 자동소멸 됩니다.
  • 원더코인으로 결제 후 취소/품절/반품으로 환불완료가 되면 사용하신 원더코인은 즉시 재 적립됩니다
  • 원더코인 사용확인은 마이페이지 > 원더코인 에서 확인 가능합니다.
  • 상품 판매가의 10% 까지 코인 사용이 가능합니다.
