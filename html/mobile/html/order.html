원더코인 사용안내

  • 원더코인은 1000 코인 이상부터 1코인 단위로 결제 시 사용 가능합니다.
  • 원더코인은 기본적으로 적립일로부터 1년 동안 사용 가능하며 이후 자동소멸 됩니다.
  • 원더코인으로 결제 후 취소/품절/반품으로 환불완료가 되면 사용하신 원더코인은 즉시 재 적립됩니다
  • 원더코인 사용확인은 마이페이지 > 원더코인 에서 확인 가능합니다.
  • 상품 판매가의 10% 까지 코인 사용이 가능합니다.
